JLPT N1 202112

2023-11-21 | 日本語

  • 錯覚（さっかく）

  • 枯渇（こかつ）

  • 克明（こくめい）：认真仔细，一丝不苟

  • 緊迫（きんぱく）

  • 圧迫（あっぱく）

  • 風習（ふうしゅう）

  • 厚かましい（あつかましい）：厚脸皮

  • あじけない（あじけない）：boring

  • もどかしい：令人着急的

  • 探索：探り求める,e.g. 犯人を探索する

  • ぐずぐず：慢吞吞的，松散的，啰嗦的

  • だらだら：滴答滴答，（话、文章）冗长的

  • めきめき：进步迅速的，显著的，著しい、めっきり

  • うらうら：心神不宁，徘徊（彷徨う、迷い）

  • もたれる：承受

  • おぼれる（溺れる）：沉溺，溺水

  • こじれる（拗れる）：悪化、複雑化 e.g. 話が拗れる・風が拗れる

  • しびれる：麻 脚がしびれる

  • 寛ぐ（くつろぐ）：舒服，轻松

  • うやむや：あいまい

  • お手上げ：🤷‍♀️没办法、どうしようもない

  • 押収（おうしゅう）：没收 e.g. 凶器を押収する

  • 望ましい：ideal

  • 本場：发源地，本地

  • 秘める（ひめる）：隐藏，隐瞒

  • 〜はどうあれ：不论什么方法

  • ～にかわって：代替前项

  • ～にかかわる：related

    • 〜にも関わらず：although

  • 〜にしても・〜にせよ・〜にしろ：ても（即使）

  • かりに：if

    • かりにても：就算　e.g. かりに今行っても、間に合わない

  • なんら…ない　＝　全くない

    • 全く　＝　まるで

  • なかなか

    • positive：unexpected results e.g. なかなか美味しいね
    • negative:（usually used with ない）不简单/不容易实现 e.g. バスがなかなか来ない

  • あまりにも：过于

あの時すぐに治療しておけばよかったものを、放っておくから悪化してしまたんだ

  • ばかりに　＝　だけに：because of (causal relationship)
    • ばかりに：bad ending
    • だけに：could be both good or bad ending
  • こと　＆　もの
    • こと：specific e.g. 先生の猫は可愛いことだ
    • もの：general e.g.　猫はみんな可愛いものだ
      • ものを　＝　のに　e.g. こんなに美味しいものを食べないの？・こんなに美味しのに、食べないの？
      • ものの：雖然但是 e.g. 知ってるものの知らな振りをする
      • ものか：強い否定 e.g. 知るもんか・知るものか

昨日行ったそば屋は、たくさんの人が並んでいたので待たされるかと思いきや、意外にもすぐに席に案内された
-　思いきや：以為，哪曾想，不料

  • ておく

    • 表狀態
    • 事先做某事

  • とすると：假設這樣做的話

  • humble form

    • お　＋　Vます　＋　します
    • ご　＋　Vします（III）　＋　します
      • します → いだします
    • 「私」は「他人」に：〜ていただく
    • 「他人」に（「私」は）：〜てくださる

  • 〜にしては：as for，對於

  • 〜からには：既然

仕事忙しくて、今年は旅行どころではない

  • 〜にすぎない：不過是

  • 〜ところではない：不是。。。的時候

  • ～にほかならない：正式（不是其他）

  • ～といっても過言ではない：。。。也不為過

  • かぎり

    • limit
      • limited: 女性に限る
      • as far as: 知ってる限りで
      • until: 本日を限りに
    • not limited to（positive）
      • adj ＋　限りだ：very
    • extreme
      • do sth at best: 力限りで
      • unfortunately：今日に限って、雨が降る
    • とは限らない: unlikely
      • 金持ちは必ず幸せとは限らない

私の夢を応援してくれた人たちに心から感謝している。特に、いつも支えてくれた両親にはどんなに感謝してもし切れない。

  • V去ます　＋　きれない：完，尽
  • ことはない：no need e.g. 心配することはない
  • 〜てたまらない：very（physically） e.g. 痛くてたまらない
  • 〜てならない：very（psychologically） e.g. 悔やんてならない
    • 〜て仕方ない
    • 〜てしょうがない
    • 〜てしようがない

バイオリンは、半年レッスンを受けても、曲が弾けるレベルにまで上達するのは難しく、せいぜい正しい持ち方ができるようになるくらいだ

  • せいぜい：尽量，尽可能的，至多，充其量

  • ようになる：客观动作变化 e.g.　地下鉄で通勤できるようになる

  • ようにする：主观动作变化 e.g.　ダイエットするようにしている

  • ことになる：客观决定 e.g.　出張することになる

  • ことにする：主观决定 e.g.　旅行することにする

  • なり

    • 一。。。就 e.g. 帰るなり、すぐ出かけてしまいました。
    • 无论 e.g. 賛成するなり、反対するなり

  • ものだ：（感叹）就是。。。
