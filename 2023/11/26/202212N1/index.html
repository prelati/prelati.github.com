このレストランのシェフはまだ二十代だが、著名な料理評論家をも満足させる腕の持ち主であると、テレビや雑誌で評判になっている

  • 評判になる：好評

  • 使役

    • A　は　B　を　Vさせる
    • A　は　B　に　C　を　Vさせる

  • 〜のついでに：順便 = がてら

    • 散歩のついてに　＝　散歩がてら

  • まもなく：可以一過去時間為基準

  • もうすぐ：只能以現在時間為基準

  • ～につき

    • 每。。。e.g. 三人につき、一つを上げる
    • causal relationship：～によって

  • date ＋　をもって：到。。。為止

高校に入学して、特に入りたいクラブもなかったので、友人に誘われるまま、テニス部に入った。

  • まま：維持現有狀態
  • ようが　＝　ても：即使
    • 疑問句 + ても / 。。。ても。。。ても：無論
    • 。。。ようが。。。まいが　＝　なくても
    • 。。。ようと。。。まいと
    • e.g. 賛成しようがするまいが
  • 〜ない　＝　〜ず　＝　〜ぬ　＝　〜まい　＝　〜ざる
  • 〜上に：表並列關係 = それに　＝　しかも：既。。。又
  • ～上は
    • Vた　＋　上は：以上は
    • e.g. 決めた上は（以上は）やるしかない
  • 〜上で
    • based on (usually used with たform) e.g. 相談した上で
    • on the aspect of V辭 / noun+の e.g.　日本語を勉強する上で役に立つものはなんですか
  • ~ない限り：只要不。。。 e.g. 法律を犯さない限り
  • 申し上げます ← 言う（humble）

学生時代はすっとサッカーをしていたのだが、社会人になってからと言うもの、運動する機会がすっかり減ってしまった。

  • 〜なってからというもの：just because

  • 見当たらない：cannot found

  • ～たら

    • 一。。。就
    • 意外情況發生 e.g. ドアが開けたら、猫を飛び出す

  • Vたつもり　/ Vているつもり：本以為，自認為

  • ～っこない：ありえない・知らない　e.g.　できっこない　＝　できない

  • わけだ：based on evidence, should be

  • わけじゃない・わけではない：並非

  • わけがない：ありえない

  • わけにはいかない：不應該，should not e.g. 明日は試験だから、遊びわけにはいかない

  • ～もんじゃない：豈止大

  • 〜もんだ

    • 感嘆：人生は早いもんだ
    • 就是。。。：人間は死ぬものだ
    • Vた　＋　もの：回憶：子供の頃よく遊んだものだ
